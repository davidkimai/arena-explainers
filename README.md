# arena-explainers

Reusable GitHub Pages site for browser-friendly study explainers.

## Structure

- `site/index.html` - landing page
- `site/<slug>/index.html` - individual explainers
- `.github/workflows/pages.yml` - GitHub Pages deployment via GitHub Actions

## Current explainers

- `mech-interp-week4` - Intro to Mechanistic Interpretability

## Publishing

Push to `main`, then enable **Settings → Pages → Source: GitHub Actions** in the GitHub repo.
